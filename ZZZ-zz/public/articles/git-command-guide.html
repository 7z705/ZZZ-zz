常见 Git 命令及使用指南

发布于 2026-04-12

Git 是前端开发中最重要的协作工具之一。掌握常见命令不仅能提升日常效率，也能减少很多因为操作失误造成的返工。

1. 初始化与克隆

git init
git clone 仓库地址

2. 查看状态与提交

git status
git add .
git commit -m "feat: 完成文章搜索功能"

3. 分支相关

git branch
git checkout -b feature/article-search
git switch main

4. 拉取与推送

git pull origin main
git push origin feature/article-search

5. 查看日志与差异

git log --oneline
git diff
git diff --staged

6. 常见建议

提交信息尽量清晰、分支命名统一、冲突尽早解决、避免在不理解的情况下使用强制覆盖命令，这是团队协作里非常重要的习惯。
