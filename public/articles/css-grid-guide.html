CSS Grid 布局完全指南

发布于 2024-06-11

CSS Grid 布局是 W3C 推出的二维布局系统，可以同时处理行和列，彻底改变了我们设计网页布局的方式。相比 Flexbox 的一维布局，Grid 更适合做整体页面框架。

1. 基本概念

Grid 容器通过 display: grid 定义，其直接子元素成为网格项。可以使用 grid-template-columns 和 grid-template-rows 定义列宽和行高。

2. 核心属性示例

.container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  gap: 20px;
}
.item {
  background: #eee;
  padding: 20px;
}

3. 常用布局模式

  • 圣杯布局：grid-template-areas: "header header" "sidebar main" "footer footer";
  • 响应式卡片网格：grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
  • 内容居中：place-items: center;

4. 结合媒体查询实现自适应

@media (max-width: 768px) {
  .container {
    grid-template-columns: 1fr;
  }
}

CSS Grid 让复杂的响应式布局变得简单直观，是现代前端开发必须掌握的核心技能。
